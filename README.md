# Bachelor-Thesis-Cybersecurity-using-Machine-learning-and-Deep-Learning-
Bachelor Thesis project focused on Cybersecurity using Machine Learning and Deep Learning techniques for intrusion detection, anomaly detection, and cyber threat analysis. The project compares ML and DL models to improve detection accuracy, reduce false positives, and enhance intelligent network security systems.



Cybersecurity using Machine Learning and Deep Learning

This repository contains the implementation, experiments, and research work for my Bachelor’s Thesis on Cybersecurity using Machine Learning and Deep Learning. The project focuses on detecting and preventing cyber threats using intelligent data-driven models capable of identifying malicious activities in network traffic and system behavior.

The primary objective of this research is to explore how Machine Learning (ML) and Deep Learning (DL) techniques can improve modern cybersecurity systems by providing faster, scalable, and more accurate threat detection compared to traditional signature-based approaches.

Features
Network intrusion detection using ML and DL models
Data preprocessing and feature engineering for cybersecurity datasets
Comparative analysis of traditional ML algorithms and deep learning architectures
Real-time or simulated attack detection
Model training, evaluation, and performance visualization
Detection of anomalies, malicious traffic, and cyber attacks
